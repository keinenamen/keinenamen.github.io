T39 组合总和

题干描述

给你一个 无重复元素 的整数数组 candidates 和一个目标整数 target ，找出 candidates 中可以使数字和为目标数 target 的 所有 不同组合 ，并以列表形式返回。你可以按 任意顺序 返回这些组合。

candidates 中的 同一个 数字可以 无限制重复被选取 。如果至少一个数字的被选数量不同，则两种组合是不同的。

对于给定的输入，保证和为 target 的不同组合数少于 150 个。

输出描述

解题思路

根据题意，想到可以采用DFS + 剪枝来解决

实现流程：

第一步：先将candidates数组按升序排列，方便后续剪枝

第二步：实现dfs操作，首先是递归终止条件：

case 1 : target == 0 ,找到符合条件，将当前path加入ansList后return

case 2 : target < 0 ,直接return，实际上在外部剪枝了的话，这里无需再判断

接着，对candidates中的每一个元素做一个dfs。另外，需要做回溯处理，不然path中会记录所有尝试过的值。

最后，对于path 的去重问题，核心是 ：每次遍历candidates时，只选择没有访问过的元素做dfs。因此对于dfs函数需要传入一个当前正在处理的元素下标，在下一轮遍历时，直接从当前下标开始遍历，使得后续加入path中的元素是之前没有记录过的。

（这种去重思想可以参考 三数之和）
