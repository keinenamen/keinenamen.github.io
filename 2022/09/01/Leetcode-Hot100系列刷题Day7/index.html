T62 不同路径

题干描述

一个机器人位于一个 m x n 网格的左上角 （起始点在下图中标记为 “Start” ）。

机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为 “Finish” ）。

问总共有多少条不同的路径？

输出描述

解题思路

这里的话，考虑用动态规划的思想来做。对于大小为mxn的矩阵，由于每次只能向下或者向右移动一步，当我们到达某个格子(i,j) ，必然是经过了格子(i-1,j) 或者 (i,j-1),对于边界情况：

第一行(i = 0)的格子(0,j)只能是上一个格子往右移动一步，即 只经过了格子(0,j-1)

第一列(j = 0)的格子(i,0)只能是上一个格子往下移动一步，即 只经过了格子(i-1,0)

我们假设 f(i,j) 为到达格子(i,j)的路径数，那么可以得到状态转移方程：f(i,j) = f(i-1,j) + f(i,j-1)

对于第一行和第一列的情况可以单独处理。

T64 最小路径和

题干描述

给定一个包含非负整数的 *m* x *n* 网格 grid ，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。

说明：每次只能向下或者向右移动一步。

输入描述

输入描述

解题思路

这道题和上一题思路基本一致，考虑用动态规划的思想来做。

对于原大小为mxn矩阵，新建一个大小为mxn的dp二维数组,dp[i][j] 表示到达grid[i][j]的最小路径和。

由于每次只能向下或者向右移动一步，当我们到达某个格子(i,j) ，必然是经过了格子(i-1,j) 或者 (i,j-1),对于边界情况：

第一行(i = 0, j > 0)的格子(0,j)只能是上一个格子往右移动一步,即dp[0][j] = dp[0][j-1]+grid[i][j]

第一列(j = 0, i > 0)的格子(i,0)只能是上一个格子往下移动一步，即 dp[i][0] = dp[i-1][0]+grid[i][j]

对于一般情况，dp[i][j] = Math.min(dp[i-1][j],dp[i][j-1])+grid[i][j]

最终返回dp[m-1][n-1]即可
